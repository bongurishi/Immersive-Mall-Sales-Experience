🚀 Immersive Mall Sales Experience (IMSE)

A cinematic, conversion-focused interactive web application designed to replace fragmented sales pitches with a single immersive experience for retail leasing, sponsorships, and event partnerships.

🎯 Overview

IMSE is not just a website—it’s a conversion engine.
It enables prospects to explore, evaluate, and take action through a video-first, interactive experience that simulates real decision-making.

⚙️ Tech Stack
React + Vite – Fast development and optimized builds
Tailwind CSS – Utility-first, fully responsive design system
Framer Motion – Smooth animations and transitions
Lenis – Premium smooth scrolling experience
Web Audio API – Lightweight, asset-free interaction sound design

✨ Key Features
🎬 Cinematic Experience
Video-first storytelling
Smooth scrolling and scroll-triggered animations
High-end, luxury-inspired UI

🎯 Conversion-Focused Design
Persistent CTAs across the experience
ROI-driven messaging and impact metrics
Lead capture for real business use-cases

🧠 User Segmentation
Tailored experience for:
Retail Brands
Event Organizers
Sponsors
State persisted via localStorage

🧩 Interactive Event Booking Module
Select event type and audience size
Smart venue recommendations
“Book This Space” CTA for conversion
🎬 Demo Mode (Guided Experience)
Automated walkthrough of key sections
Simulates real user interaction
Ideal for presentations and quick demos

📊 Analytics Layer
Tracks:
CTA clicks
User segment selection
Event interactions
Built as a mock service (extensible to GA/Mixpanel)

🔊 Interaction Sound Design
Built using Web Audio API (no external assets)
Subtle hover and click feedback
Includes sound toggle for user control

📱 Production-Ready UX
Fully responsive (desktop + mobile)
Optimized touch targets
Accessibility considerations (reduced motion, performance fallback)

🏗️ Architecture Overview
App.tsx → Initializes smooth scrolling and renders sections
Navigation → Sticky, responsive navigation with active tracking
Hero → Video background + user segmentation
EventBooking → Interactive decision-making module
DemoMode → Automated guided experience
Analytics Service → Tracks user interactions
